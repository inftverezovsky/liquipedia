# Project scope

## Current project name

`liquipedia`

Это отдельный проект. Он не является частью Onyx.

## First discipline

`Dota 2`

## Main product flow

```text
Open Dota 2 → enter approximate tournament name → find candidates → select page → load data → view structured result
```

## Included in MVP

- Поиск чемпионата.
- Список найденных страниц.
- Ручная загрузка выбранного турнира.
- Raw snapshot.
- Карточка турнира.
- Участники выбранного турнира.
- Матчи выбранного турнира.
- История загрузок.
- Настройки API.
- Debug/raw view.

## Excluded from MVP

- Игроки.
- Составы.
- Трансферы.
- Изменения страниц.
- Сигналы.
- Diff-мониторинг.
- Background sync.
- Обход всех чемпионатов.
- HTML scraping.
- Betting odds.
- ML predictions.
